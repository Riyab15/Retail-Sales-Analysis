# Retail Sales Analysis Project

## Project Overview

This project analyzes an online retail dataset using Python, SQL, Pandas, Matplotlib, and Seaborn.

## Project Phases

### Phase 1: Data Collection and Database Setup
- Dataset imported from CSV
- SQL database schema designed
- Table creation queries written

### Phase 2: Data Cleaning and Preparation
- Removed missing values
- Removed duplicate records
- Converted date columns
- Created TotalSales column

### Phase 3: Data Analysis
- Top selling products
- Country-wise sales analysis
- Monthly sales trends
- Customer purchasing patterns

### Phase 4: Reporting and Visualization
- Monthly Sales Chart
- Top Products Chart
- Country Sales Chart
- Correlation Heatmap
- Time Series Analysis

## Technologies Used

- Python
- Pandas

- Matplotlib
- Seaborn
- SQL
