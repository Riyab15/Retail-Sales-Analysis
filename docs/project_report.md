# Retail Sales Analysis Report

## Introduction

The objective of this project is to analyze retail sales data and identify customer purchasing patterns.

## Data Cleaning

- Missing values removed
- Duplicate records removed

## Data Preparation

- Date conversion performed
- TotalSales column created

## Analysis

- Top Products
- Country-wise Sales
- Monthly Sales Trends

## Visualization

- Monthly Sales Chart
- Top Products Chart
- Country Sales Chart
- Heatmap
- Time Series Analysis

## Conclusion

The project successfully identified key sales trends, customer behavior patterns, and high-performing products.
